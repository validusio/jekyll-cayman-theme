Deep Learning AMI

Accelerate GPU-based workflows on Amazon Web Services

Spend less time compiling, installing, and configuring Machine Intelligence tools. The Validus Deep Learning Amazon Machine Image (AMI) is pre-packaged with popular machine learning frameworks:

Getting Started

The Deep Learning AMI is available on the AWS Marketplace: {{ site.aws_marketplace_url }}

Latest versions of pip packages of popular deep learning frameworks come pre-installed. Also included:

  • Apache MXNet
  • Caffe2
  • TensorFlow
  • PyTorch
  • Keras
  • CNTK
  • Theano
  • Anaconda Data Science packages
The AMI is designed to provide a stable, secure, and high performance execution environment for running deep learning applications on Amazon EC2. The Deep Learning AMI is free to try for 10 days. During this trial there will be no hourly charges for the instance. However, AWS infrastructure charges still apply. Free Trials will automatically convert to paid hourly subscription upon expiration.
